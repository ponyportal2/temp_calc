This is a smart calculator like the one you get in Google when typing math expressions in the search. It has three modes of operation:

Simple

With X parameter

Binary Search expression solver.

--------------------------------

1. Type your expression here. You can use x, but if you do - you should provide a correct x value (can be an expression) in field 2. Important - in this calculator division has precedence over multiplication, use brackets to specify desired order of operations.

USE LEFT ARROW KEY TO DELETE WHOLE WORDS OR NUMBERS IN THIS FIELD

You will get calculation results and solver answers in field 1, so it acts as both an input and an output field.

USE Option+Backspace TO CLEAN FIELD 1

2. Type your x here (optional).

USE Shift+Backspace TO CLEAN FIELD 2

3. Expected answer for an expression solver (optional).

If there's anything in field 3, then SmartCalc will work in expression solver mode.

USE \(Backslash)+Backspace TO CLEAN FIELD 3

4. Equals button, press it or an Enter key to see the result in the input field (field 1).

5. The graph of a function with 1000000 scale, will only get drawn if x is supplied (field 2).

For the graph divide the expression by desired amount to get "zoom", for example try dividing it by 1000000, or multiplying by a small number, like 0.00001.

--------------------------------

Allowed operations for the calculator are: + - * / ^ mod cos() sin() tan() acos() asin() atan() sqrt() ln() log(). Unary + and - are allowed. You can use brackets.

IMPORTANT: use * symbol instead of implicit multiplication like 5x or 5(x+2). Add the multiplication sign to avoid errors. For example: 5*x, 5*(x+2).
